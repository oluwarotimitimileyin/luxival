How we work

Common questions

What you might be wondering

Do I need technical knowledge to use an AI agent system?
No. We design the system so that you interact with it through a simple interface — usually a form, a dashboard, or a messaging channel. The technical complexity lives inside the agent, not in your daily workflow.
How much does it cost to run an agent after it is built?
The main ongoing cost is Claude API usage, which scales with how much the agent runs. For a typical content or lead agent running daily, costs are usually between €10 and €50 per month depending on volume.
Can the agent connect to tools I already use?
Yes. We can integrate with Supabase, Google APIs, Slack, email, Notion, Airtable, REST APIs, and most services that have a public API or webhook.
What if the agent makes a mistake?
All agents are built with error handling, confidence thresholds, and human-in-the-loop checkpoints for high-stakes decisions. You define what the agent does autonomously and what it escalates to you.
How long does a build take?
A Starter Agent takes roughly one to two weeks. A Multi-Agent Pipeline takes three to five weeks. Enterprise systems are scoped individually.
