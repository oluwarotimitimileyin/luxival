# Luxival Launch Checklist

## 1. Deployment and Hosting
- [ ] Initialize Git repository and commit current site files
- [ ] Push repository to GitHub or another Git provider
- [ ] Deploy the site to Vercel (recommended) or Netlify
- [ ] Confirm preview deployment URL works
- [ ] Confirm all pages load without HTML/CSS/JS errors

## 2. Domain and HTTPS
- [ ] Add `luxival.com` and `www.luxival.com` to hosting provider domain settings
- [ ] Update DNS records at your domain registrar
  - `A` record for `@` to hosting provider IP or use nameservers
  - `CNAME` record for `www` to hosting provider DNS target
- [ ] Wait for DNS propagation and domain verification
- [ ] Confirm HTTPS is enabled and secure on both `https://luxival.com` and `https://www.luxival.com`

## 3. SEO and Search Engine Verification
- [ ] Add `sitemap.xml` at site root
- [ ] Add `robots.txt` at site root
- [ ] Confirm each page has a unique title and meta description
- [ ] Confirm Open Graph and Twitter card metadata is present
- [ ] Submit sitemap to Google Search Console
- [ ] Verify property in Google Search Console
- [ ] Submit sitemap to Bing Webmaster Tools

## 4. Site Content and UX
- [ ] Confirm homepage hero and positioning copy are polished
- [ ] Confirm services page clearly lists all digital and transport services
- [ ] Confirm tourism page includes pricing calculator and booking CTA
- [ ] Confirm portfolio page structure is ready for case studies
- [ ] Confirm about page uses actual founder background and credentials
- [ ] Confirm contact page includes a functional inquiry flow
- [ ] Confirm all CTAs link to contact or service pages

## 5. Technical QA
- [ ] Test navigation across all pages
- [ ] Test responsive behavior on mobile, tablet, and desktop
- [ ] Test the pricing calculator flow
- [ ] Test contact form submission or placeholder instructions
- [ ] Check for broken links and missing assets
- [ ] Confirm no console errors in browser DevTools

## 6. Launch Preparation
- [ ] Create `privacy.html` placeholder
- [ ] Create `terms.html` placeholder
- [ ] Create `sitemap.xml` and `robots.txt` if not already present
- [ ] Add any required business contact or support details
- [ ] Prepare launch announcement copy or social plan

## 7. Post-launch Monitoring
- [ ] Monitor Google Search Console coverage and indexing
- [ ] Monitor analytics and traffic if enabled
- [ ] Verify form inquiries are delivered correctly
- [ ] Review site speed and performance periodically
- [ ] Schedule an update for case studies and portfolio content
